picedit v1.3m2

Color Panel

  • Right click: Choose colour for visual screen.
  • Left click: Choose colour for priority screen.
  • Off: Disable visual/priority drawing.

Features

  • Click on buffer position field to enter a specific position.
  • Use navigation buttons to navigate through picture code buffer.
  • Drawing actions can be inserted anywhere inside the picture.
  • The delete button removes the drawing action that is currently being shown in the data box (bottom right corner).
  • The brush tool is set up using the buttons immediately to the right of the off button.
  • Right clicking on the picture surface once stops current tool.
  • Right clicking on the picture surface twice switches tool to None.
  • Use the background image option to enable tracing of PNG, JPG, etc.

Shortcut Keys

TABSwitch between visual/priority screens.
L, l, F1Activates line tool
P, p, F2Activates pen tool
S, s, F3Activates step tool
F, f, F4Activates fill tool
B, b, F5Activates brush tool
F6Toggles background picture
HomeStart of picture code buffer
Left arrowBack one drawing action
Right arrowForward one drawing action
EndEnd of picture code buffer
DelRemove drawing action at current buffer position
