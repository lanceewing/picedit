Welcome

PICEDIT is a picture editor for creating and editing pictures for the Sierra On-line 3D animated adventure game system known as AGI, the system used to create such games as King's Quest, Police Quest and Space Quest.

Features

  • Use navigation keys to navigate through picture code buffer, or simply drag the slider.
  • Alternatively use the left hand side Commands list to navigate.
  • Drawing actions can be inserted anywhere inside the picture.
  • The delete button removes the next drawing action.
  • Right clicking on the picture surface once stops current tool.
  • Right clicking on the picture surface twice switches tool to None.
  • Use the background image option to enable tracing of PNG, JPG, etc.
  • To add a single point line, simply click on the start point then right-click to stop drawing.
  • Turn on grid lines (bands) to see where each priority band starts and ends.
  • Dual visual/priority mode that shows priority bands through a transparent visual screen.
  • Use mouse wheel to zoom in and out or alternatively use the plus and minus keys.
  • Use mouse wheel button as a short cut to bring up the colour chooser.
  • Open and edit multiple pictures at the same time.
  • Turn on Ego Test mode to walk ego around the screen. Feels just like in a game.
